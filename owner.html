OWNER COMMANDS

System Control

Restart Bot

Reboots the entire bot system to clear cache and fix minor lag issues.

.restart

Public / Private Mode

Switch bot access between all users or only the owner.

.mode public/private
User Management

Add Premium

Give a specific user unlimited access to bot features and high-speed downloads.

.addprem [number]

Broadcast Message

Send a notification message to all groups and individual chats at once.

.bc [text]
